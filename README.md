# Minimalist Claude Telegram Bot

The bare necessities you need to connect Claude to Telegram. No extra BS.

## Why this exists

Most frameworks and connectors out there — OpenClaw, various Telegram bot frameworks, and other "all-in-one" solutions — add way too much complexity. Extra layers, opinionated abstractions, and bloated dependencies that get in your way more than they help.

This is different. This is just the minimal wiring you need to get Claude talking through Telegram. Nothing more.

## What you get

- A clean, simple connection between Claude and Telegram
- No unnecessary abstractions or framework lock-in
- A starting point you actually understand, so you can build your own system on top of it

## Philosophy

Start simple. Understand every line. Build what YOU need from there.

If you want a framework that thinks for you, look elsewhere. If you want a foundation you can own and extend — start here.
